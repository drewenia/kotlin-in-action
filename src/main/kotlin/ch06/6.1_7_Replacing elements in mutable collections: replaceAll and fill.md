# Replacing elements in mutable collections: replaceAll and fill

Genel olarak, functional programming tarzı immutable collection’larla çalışmanızı teşvik eder; ancak yine de mutable
collection’larla çalışmanın daha ergonomik olduğu bazı durumlar olabilir. Bu tür durumlar için, Kotlin standard library,
collection’larınızın içeriğini değiştirmenize yardımcı olacak birkaç convenience function ile birlikte gelir.

MutableList’e uygulandığında, replaceAll function list’teki her bir element’i, belirttiğiniz lambda’dan elde edilen
sonuçla değiştirir. Mutable list’teki tüm element’leri aynı value ile değiştirme gibi özel bir case için ise, kısa yol
olarak fill function’ı kullanabilirsiniz. Bu örnekte, önce input collection’daki tüm element’leri onların uppercase
varyantlarıyla değiştiriyor, ardından tüm isimleri placeholder metinle değiştiriyorsunuz:

```kotlin
fun main() {
    val names = mutableListOf("Martin", "Samuel")
    names.replaceAll { it.uppercase() }
    println(names) // [MARTIN, SAMUEL]

    names.fill("Redacted")
    println(names) // [Redacted, Redacted]
}
```
