# Declaring functions that return or receive other functions: Higher-order functions

Bu chapter’ın temel yeni fikri, higher-order function concept’idir. Tanım gereği, higher-order function başka bir
function’ı argument olarak alan veya bir function return eden function’dır. Kotlin’de function’lar, lambda’lar
kullanılarak ya da function reference’ları aracılığıyla value olarak represent edilebilir. Dolayısıyla, bir lambda’yı
veya bir function reference’ını argument olarak geçebildiğin herhangi bir function, ya da bunlardan birini return eden
bir function, ya da her ikisini birden yapan bir function higher-order function’dır. Örneğin, filter standard library
function’ı bir predicate function’ı argument olarak alır ve bu nedenle bir higher-order function’dır:

```kotlin
list.filter { x > 0 }
```

Chapter 6’da, Kotlin standard library’de declare edilmiş map, with vb. birçok başka higher-order function gördün. Şimdi,
bu tür function’ları kendi code’un içinde nasıl declare edebileceğini öğreneceksin. Bunu yapmak için önce function
type’larıyla tanıştırılman gerekir.
