# Entry

Chapter 5'de lambda’larla tanıştırıldın; burada genel concept’i inceledin ve 6. chapter’da lambda kullanan standard
library function’larına daha derinlemesine daldın. Lambda’lar abstraction’lar oluşturmak için harika bir araçtır ve
elbette güçleri collection’lar ve standard library’deki diğer class’larla sınırlı değildir. Bu chapter’da, lambda’ları
argument olarak alan ya da onları return eden kendi function’ların olan higher-order function’ları nasıl create
edeceğini öğreneceksin. Higher-order function’ların code’unu nasıl simplify edebileceğini, code duplication’ı nasıl
ortadan kaldırabileceğini ve güzel abstraction’lar nasıl build edebileceğini göreceksin. Ayrıca, lambda kullanımıyla
ilişkili performance overhead’ini ortadan kaldıran ve lambda’lar içinde daha esnek control flow sağlayan güçlü bir
Kotlin feature olan inline function’larla da tanışacaksın.
