# Creating ranges from objects: The rangeTo and rangeUntil conventions

Bir range oluşturmak için `..` syntax’ını kullanırsınız: örneğin, 1..10 ifadesi 1’den 10’a kadar olan sayıları temsil
eder. Range’lerle 2. bölümde karşılaştınız, ancak şimdi bir range oluşturmanıza yardımcı olan convention’ı
tartışalım. `..` operator’ü, rangeTo function’unu çağırmanın kısa bir yoludur:

![img_11.png](assets/img_11.png)

rangeTo function’u bir range döndürür. Bu operator’ü kendi class’ınız için define edebilirsiniz. Ancak class’ınız
Comparable interface’ini implement ediyorsa, buna gerek yoktur: Kotlin standart kütüphanesi aracılığıyla herhangi bir
comparable element için bir range oluşturabilirsiniz. Kütüphane, herhangi bir comparable element üzerinde çağrılabilen
rangeTo function’unu define eder:

```kotlin
operator fun <T : Comparable<T>> T.rangeTo(that: T): ClosedRange<T>
```

Bu function, farklı element’lerin range’e ait olup olmadığını kontrol etmenizi sağlayan bir range döndürür. Örnek
olarak, LocalDate class’ını (Java 8 standart kütüphanesinde tanımlıdır) kullanarak bir tarih range’i oluşturalım.

```kotlin
fun main() {
    val now = LocalDate.now()
    // Şu andan başlayan 10 günlük bir range oluşturur.
    val vacation = now..now.plusDays(10)

    println(vacation) // 2026-01-11..2026-01-21

    // Belirli bir tarihin range’e ait olup olmadığını kontrol eder.
    println(now.plusWeeks(1) in vacation) // true
}
```

`now..now.plusDays(10)` expression'ı, compiler tarafından `now.rangeTo(now.plusDays(10))` şeklinde dönüştürülür. rangeTo
function’u LocalDate’in bir member’ı değil, daha önce gösterildiği gibi Comparable üzerinde bir extension function’dır.
rangeTo operator’ü, aritmetik operator’lerden daha düşük önceliğe sahiptir. Ancak karışıklığı önlemek için,
argument’leri parantez içinde kullanmak daha iyidir:

```kotlin
fun main() {
    val n = 9
    println(0..(n + 1)) // 0..10
}
```

Ayrıca, `0..n.forEach {}` expression'ının compile olmayacağını unutmayın; çünkü bir method’u çağırmak için range
expression'ı parantez içine almanız gerekir:

```kotlin
fun main() {
    val n = 9
    // Bir method’u çağırmak için range’i parantez içine alın.
    (0..n).forEach { print(it) } // 0123456789
}
```

rangeTo operator’ünün analogu olan rangeUntil operator `(..<)`, belirtilen üst sınırı içermeyen açık uçlu bir range
döndürür:

```kotlin
fun main() {
    (0..<9).forEach { print(it) } // 012345678
}
```

Şimdi, conventions’ın bir collection veya range üzerinde nasıl iterate etmenize izin verdiğini tartışalım.
