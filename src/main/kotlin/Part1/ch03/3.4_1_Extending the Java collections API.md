# Extending the Java collections API

Java library class’larının instance’ları olmalarına rağmen Kotlin’de collection’larla out of the box bu kadar çok
şey yapabilmenin neden mümkün olduğu. Artık cevap açık olmalı: last ve sum function’ları extension function olarak
declared edilmiştir ve Kotlin file’larında her zaman default olarak import edilirler! Last function, önceki bölümde ele
alınan String için lastChar’dan daha karmaşık değildir; List class’ı üzerinde bir extension’dır. Sum için ise
sadeleştirilmiş bir declaration gösteriyoruz (gerçek library function yalnızca Int sayılar için değil, her türlü number
type için çalışır):

```kotlin
fun <T> List<T>.last(): T { /* returns the last element */
}
fun Collection<Int>.sum(): Int { /* sum up all elements */
}
```

Kotlin standard library’de çok sayıda extension function declared edilmiştir ve bunların hepsini burada
listelemeyeceğiz. Kotlin standard library’deki her şeyi öğrenmenin en iyi yolunun ne olduğunu merak edebilirsin.
Buna gerek yok—bir collection ya da başka bir object ile ilgili bir şey yapman gerektiğinde, IDE’deki code completion
sana o type’taki object için kullanılabilir tüm function’ları gösterir. Bu liste hem regular method’ları hem de
extension function’ları içerir; ihtiyacın olan function’ı seçebilirsin.
