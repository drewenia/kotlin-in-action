# Working with strings and regular expressions

Kotlin string’leri Java string’leriyle tamamen aynıdır. Kotlin code’unda oluşturulan bir string’i herhangi bir Java
method’una geçebilirsin ve Java code’undan aldığın string’ler üzerinde Kotlin standard library method’larının herhangi
birini kullanabilirsin. Herhangi bir conversion söz konusu değildir ve ek wrapper object’ler oluşturulmaz. Kotlin,
standart Java string’leriyle çalışmayı daha keyifli hale getirmek için birçok kullanışlı extension function sağlar.
Ayrıca, bazı kafa karıştırıcı method’ları gizler ve daha anlaşılır olan extension’lar ekler. API farklılıklarına dair
ilk örnek olarak, Kotlin’in string’leri bölmeyi nasıl ele aldığına bakalım.
