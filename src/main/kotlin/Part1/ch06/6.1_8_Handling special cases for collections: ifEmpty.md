# Handling special cases for collections: ifEmpty

Çoğu zaman, bir programın ancak input collection empty değilse devam etmesi anlamlıdır — böylece process edilecek gerçek
element’ler vardır. ifEmpty function ile, collection herhangi bir element içermediği durumda bir default value üreten
bir lambda sağlayabilirsiniz:

```kotlin
fun main() {
    val emptyList = emptyList<String>()
    val full = listOf("apple", "orange", "banana")

    println(emptyList.ifEmpty { listOf("no", "values", "here") }) // [no, values, here]
    println(full.ifEmpty { listOf("no", "values", "here") }) // [apple, orange, banana]
}
```

### ifBlank: ifEmpty’s sibling function for strings

Text ile çalışırken (ki bunu çoğu zaman character’lardan oluşan bir collection olarak da ele alırız), bazen “empty”
gereksinimini “blank” olarak gevşetiriz: yalnızca whitespace character’lar içeren string’ler, saf empty string’lerden
nadiren daha anlamlıdır. Bunlar için bir default value üretmek amacıyla ifBlank function’ı kullanabiliriz:

```kotlin
fun main() {
    val blankName = " "
    println(blankName.ifEmpty { "(unnamed)" }) //
    println(blankName.ifBlank { "(unnamed)" }) // unnamed
}
```
