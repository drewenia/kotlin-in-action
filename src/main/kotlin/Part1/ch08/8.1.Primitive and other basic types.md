# Primitive and other basic types

Bu bölüm, Int, Boolean ve Any gibi programlarda kullanılan basic type’ları açıklar. Java’dan farklı olarak Kotlin,
primitive type’lar ile wrapper’lar arasında ayrım yapmaz. Bunun nedenini ve bunun perde arkasında nasıl çalıştığını
birazdan öğreneceksin. Ayrıca Kotlin type’ları ile Object ve Void gibi Java type’ları arasındaki karşılığı da
göreceksin.
