# Collections and arrays

Zaten çeşitli collection API’larını kullanan birçok kod örneği gördün ve 3.1 bölümünde tartışıldığı gibi, Kotlin Java
collections library üzerine inşa edilmiştir ve extension function’lar aracılığıyla eklenen özelliklerle
zenginleştirilmiştir. Kotlin’deki collection desteğinin ve Java ile Kotlin collection’ları arasındaki karşılığın
hikayesi daha fazlasını içerir ve şimdi detaylara bakmak için iyi bir zaman.
